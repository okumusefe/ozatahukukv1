# Özata Hukuk - Avukat Websitesi

Modern ve mobil uyumlu bir avukat websitesi.

## Özellikler

- ✅ Tamamen mobil uyumlu (responsive) tasarım
- ✅ Modern ve profesyonel görünüm
- ✅ Kolay navigasyon
- ✅ Hızlı yükleme
- ✅ SEO dostu yapı

## Sayfalar

1. **Anasayfa** (index.html) - ✅ Tamamlandı
   - Hero bölümü
   - Giriş metni
   - Neden Özata Hukuk bölümü
   - Çalışma alanları özeti
   - Alıntı bölümü
   - İletişim çağrısı

2. **Hakkımızda** (hakkimizda.html) - ⏳ İçerik bekleniyor
3. **Çalışma Alanlarımız** (calisma-alanlari.html) - ⏳ İçerik bekleniyor
4. **Makaleler** (makaleler.html) - ⏳ İçerik bekleniyor
5. **İletişim** (iletisim.html) - ⏳ İçerik bekleniyor

## Kullanım

Websitesini görüntülemek için `index.html` dosyasını bir web tarayıcısında açın.

### Yerel Sunucu ile Çalıştırma (Önerilen)

Python 3 yüklüyse:
```bash
python3 -m http.server 8000
```

Ardından tarayıcıda `http://localhost:8000` adresini ziyaret edin.

## Sonraki Adımlar

Diğer sayfaların içeriklerini eklemek için lütfen aşağıdaki bilgileri sağlayın:

1. Hakkımızda sayfası içeriği
2. Çalışma alanları detaylı açıklamaları
3. Makaleler listesi
4. İletişim bilgileri (adres, telefon, e-posta, harita)
